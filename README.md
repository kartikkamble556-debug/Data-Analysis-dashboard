# Data-Analysis-dashboard
This Tableau project analyzes sales data to evaluate revenue, profit, regional performance, and product trends. Interactive dashboards, KPIs, and visualizations help identify growth patterns, high-performing regions, and low-profit areas, supporting data-driven decision-making and strategic business planning.
